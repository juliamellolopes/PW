Quem Somos!

Agradeço des de já a seu acesso a nossa cominidade. Este site foi criado por uma otako para outro para que você possa estar acompanhando as novas nividades e até mesmo estar conhecendo antigos sucessos. Criado pela aluno de programação web, Júlia Mello Lopes, que misturou um trabalho com algo que gosta e viu que seria interessante a criação desse website por já ter precissado de um para escolher um novo anime para assistir. Como tambem a necessidade de um local para saber as datas de lançamentos semanais dos epsodios para poder acompanhar. Então com o pensamento de que não seria a unica a ter essas mesmas necessidades eu decidi fazer esse site.
